# Resource Service
* リソース管理サービス


## Overview
* ResourceApi
    * ResourceFlavorの管理
    * Resourceの管理
* ResourceController
    * ResourceをResourceAgentにアサインする
* ResourceAgent
    * Nodeごとに複数のProviderサポートし、アサインされたResourceを実体化し、状態を管理する
    * Providerを利用してノード自身を監視し、イベントがあればAlertをMonitorControllerに通知する
        * またメトリクスをメトリクスDBに送信する
